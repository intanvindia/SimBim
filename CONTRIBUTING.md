# 🤝 Panduan Berkontribusi untuk SIMBIM

Terima kasih atas minat Anda untuk berkontribusi pada proyek SIMBIM! Kami sangat menghargai setiap bantuan dari komunitas, baik itu berupa laporan bug, saran fitur, maupun kontribusi kode.

## 📜 Kode Etik (Code of Conduct)

Untuk menjaga lingkungan yang positif dan inklusif, kami berharap semua kontributor mengikuti standar perilaku yang saling menghormati. Harap bersikap ramah dan profesional dalam semua interaksi.

## 💡 Cara Berkontribusi

Ada banyak cara untuk berkontribusi pada proyek ini:

### 🐞 Melaporkan Bug
Jika Anda menemukan bug atau perilaku yang tidak diharapkan, silakan buka **Issue** baru di [halaman Issues GitHub](https://github.com/intanvindia/SimBim/issues).

Pastikan untuk menyertakan:
-   Deskripsi yang jelas dan ringkas tentang bug tersebut.
-   Langkah-langkah untuk mereproduksi bug.
-   Perilaku yang diharapkan vs. perilaku yang sebenarnya terjadi.
-   Screenshot atau video (jika memungkinkan).

### ✨ Mengajukan Fitur Baru
Jika Anda memiliki ide untuk fitur baru atau perbaikan, kami akan senang mendengarnya! Silakan buka **Issue** baru dan jelaskan ide Anda secara detail.

## 👩‍💻 Mengirimkan Perubahan (Pull Request)

Jika Anda ingin berkontribusi langsung pada kode, ikuti langkah-langkah berikut:

1.  **Fork Repositori:**
    Buat *fork* dari repositori `intanvindia/SimBim` ke akun GitHub Anda.

2.  **Clone Fork Anda:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/SimBim.git
    cd SimBim
    ```

3.  **Buat Branch Baru:**
    Buat *branch* baru untuk perubahan Anda. Gunakan nama yang deskriptif (misalnya, `fix/bug-login` atau `feature/tambah-grafik-baru`).
    ```bash
    git checkout -b nama-branch-anda
    ```

4.  **Lakukan Perubahan:**
    Buat perubahan pada kode di *branch* Anda. Pastikan kode Anda mengikuti gaya dan standar yang sudah ada.

5.  **Commit Perubahan Anda:**
    Tulis pesan *commit* yang jelas dan informatif.
    ```bash
    git commit -m "feat: Menambahkan fitur X yang luar biasa"
    ```

6.  **Push ke Branch Anda:**
    Unggah perubahan Anda ke repositori *fork* Anda.
    ```bash
    git push origin nama-branch-anda
    ```

7.  **Buka Pull Request (PR):**
    Buka halaman repositori `intanvindia/SimBim` di GitHub dan klik tombol "New pull request". Pilih *branch* Anda sebagai sumber dan `main` sebagai target. Berikan judul dan deskripsi yang jelas untuk PR Anda.

Tim kami akan meninjau PR Anda sesegera mungkin. Terima kasih telah berkontribusi!
